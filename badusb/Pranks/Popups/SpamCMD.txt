REM Title: CMD Window Spam
REM Description: Rapidly opens many CMD windows to fill the screen
REM Target OS: Windows 10/11

GUI r
DELAY 600
STRING powershell -NoP -NonI -W Hidden -Exec Bypass

DELAY 1200

ENTER

STRING exit
